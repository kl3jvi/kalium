Conversation

data class Conversation(val id: ConversationId, val name: String?, val type: Conversation.Type, val teamId: TeamId?, val protocol: Conversation.ProtocolInfo, val mutedStatus: MutedConversationStatus, val removedBy: UserId?, val lastNotificationDate: <Error class: unknown class>?, val lastModifiedDate: <Error class: unknown class>?, val lastReadDate: <Error class: unknown class>, val access: List<Conversation.Access>, val accessRole: List<Conversation.AccessRole>, val creatorId: String?, val receiptMode: Conversation.ReceiptMode, val messageTimer: <Error class: unknown class>?, val userMessageTimer: <Error class: unknown class>?, val archived: Boolean, val archivedDateTime: <Error class: unknown class>?, val mlsVerificationStatus: Conversation.VerificationStatus, val proteusVerificationStatus: Conversation.VerificationStatus, val legalHoldStatus: Conversation.LegalHoldStatus)

A conversation is a group of users that can exchange messages.

Parameters

id

the conversation id

name

the conversation name

type

the conversation Type

teamId

the team id if the conversation was created on a team, null if there is no team

protocol

the ProtocolInfo for the communication between users

mutedStatus

the MutedConversationStatus informing about the conversation muting settings

removedBy

the user id of the user that removed the conversation if self user was removed, null otherwise

lastNotificationDate

the last notification date for the conversation, null if there are no notifications yet

lastModifiedDate

the last modified date for the conversation, null if there are no modifications yet

lastReadDate

the self user's last read date for the conversation

access

the list of available Access types that can be used to invite users to the conversation

accessRole

the list of available AccessRole user types that can be members of the conversation

creatorId

the user id of the user that created the conversation

receiptMode

the ReceiptMode flag indicating whether the conversation has read receipts enabled or not

messageTimer

the enforced timer by group conversation for self deleting messages, always NULL for 1on1 conversation

userMessageTimer

the timer duration picked by the user in absence of team or conversation enforced timers
