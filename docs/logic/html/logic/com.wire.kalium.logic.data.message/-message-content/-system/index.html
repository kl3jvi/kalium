System

sealed class System : MessageContent

Messages that are not sent between client, but are instead generated by this client and stored locally to be displayed and in-lined with the rest of the content of a conversation. These messages can be created based on events received from the backend (e.g. users added/removed from a conversation), or our own logic (e.g. missed call).
